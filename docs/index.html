What is this?

This site allow you to check if the screen you're using is an OLED or Micro LED panel. Pure black pixels on OLED and Micro LED screen do not emit light. If you see any light coming from the screen when viewing this page in the dark, it is likely that the screen you're view this on is NOT an OLED or Micro LED screen.
